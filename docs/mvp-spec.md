 # MVP 화면 설계
 
 제품명: 나만의 청음 기호 + 오선지 기반 청음 메모지 웹
 
 ---
 
 ## 1) 정보 구조(IA)
 
 - 홈
 - 새 노트
 - 에디터 (MVP 핵심)
 - 내보내기
 - 저장된 노트 목록 (로컬)
 
 ---
 
 ## 2) 핵심 화면: 에디터
 
 ### 2.1 레이아웃 (데스크톱 기준)
 
 - 상단 바
   - 문서 제목
   - 저장 상태 (자동 저장 / 저장됨)
   - 내보내기 버튼 (PNG)
 - 좌측 도구 모음
   - 기본 청음 기호 팔레트
   - 기호 크기 / 색상 (간단 프리셋)
 - 중앙 캔버스
   - 오선지 라인 + 가사 라인 렌더링
   - 무한 스크롤
 - 우측 패널 (간단 정보)
   - 선택한 기호 속성
   - 삭제 버튼
 
 ---
 
 ### 2.2 사용자 플로우 (MVP)
 
 1. 홈에서 "새 노트" 클릭
 2. 가사 붙여넣기
 3. 자동 줄 분리 + 오선지 생성
 4. 기호 드래그 앤 드롭
 5. 자동 저장
 6. PNG 내보내기
 
 ---
 
 ## 3) 화면별 상세
 
 ### 3.1 홈
 
 - 제품 설명 한 줄
 - 새 노트 시작 버튼
 - 최근 노트 리스트 (로컬)
 
 ### 3.2 새 노트
 
 - 제목 입력
 - 가사 붙여넣기 텍스트 영역
 - "에디터로 이동" 버튼
 
 ### 3.3 에디터
 
 - 가사 라인별 오선지 배치
 - 단어/음절 단위 선택
 - 기호 드래그 앤 드롭
 - 선택 기호 스타일 변경 (크기, 색)
 - 저장 상태 표시
 
 ### 3.4 내보내기
 
 - PNG 다운로드 버튼
 - 파일명 미리보기
 
 ---
 
 ## 4) MVP 인터랙션 정의
 
 - 드래그 시작: 기호 복제 + 커서 따라다니기
 - 드롭 대상: 가사 단어/음절 위
 - 자동 스냅: 드롭 시 가장 가까운 단어 중심에 고정
 - 선택: 클릭 시 선택 상태 표시 + 우측 패널 표시
 - 삭제: Backspace 또는 삭제 버튼
 
 ---
 
 ## 5) MVP 범위 스코프 체크
 
 포함
 - 가사 붙여넣기
 - 오선지 자동 배치
 - 기본 청음 기호 세트
 - 기호 드래그 앤 드롭
 - 저장 (로컬 자동 저장)
 - PNG 내보내기
 
 제외
 - 커스텀 기호
 - PDF 내보내기
 - 클라우드 동기화
 - 협업
